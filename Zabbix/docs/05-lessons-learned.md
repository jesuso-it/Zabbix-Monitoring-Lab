# Lecciones aprendidas

## Aprendizajes principales

- La monitorización básica por ICMP permite detectar rápidamente pérdida de conectividad.
- Zabbix Agent permite obtener métricas más completas del sistema.
- Un dashboard sencillo puede mejorar la visibilidad operativa.
- Las alertas deben ser claras y accionables.
- Monitorizar demasiadas cosas sin criterio puede generar ruido.

## Beneficio en un entorno real

Una solución de monitorización ayuda al equipo IT a pasar de un enfoque completamente reactivo a uno más preventivo.

## Próximas mejoras

- Añadir notificaciones por correo.
- Monitorizar servicios específicos.
- Añadir SNMP para dispositivos de red.
- Crear mapas de red.
- Automatizar el alta de hosts.
